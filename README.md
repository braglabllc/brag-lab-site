# Brag Lab — Custom Site (V2)

## Edit (2 minutes)
Open `components/siteConfig.ts` and update:
- bookingUrl
- contactEmail

## Deploy to Vercel
Vercel → Import Git Repository → select repo → Deploy.

## Key structure requirement
At repo root you must have:
- app/ (folder)
- components/ (folder)
- package.json

When you click `app` on GitHub, the URL should include `/tree/` (NOT `/blob/`).
